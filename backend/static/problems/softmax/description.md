1次元リスト $x$ が与えられます。

$softmax(x)$ を計算する関数を実装し、入力 $x$ に対して $softmax(x)$ を返してください。


## 制約



## 入力
```plaintext
x
```

- $x$ は1次元numpy配列で
```python3
np.array([...])
```
という形で書かれています。

したがって以下のようなコードで$x$を定義することができます。

```python3
x = eval(input())
```
## 出力
$softmax(x)$ を1次元numpy配列として出力してください。



## サンプル

### サンプル1

#### 入力
```plaintext
np.array([1,2,3])
```

#### 出力
```plaintext
[0.09003057, 0.24472847, 0.66524096]
```
